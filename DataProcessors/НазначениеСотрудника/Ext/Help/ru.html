Назначение сотрудника

Данная обработка служит для назначения дизайнера, ведущего конструктора и менеджера отделтным комплектам и композициям.


Кнопка сохранить   Проект и дата   Назначение ответственного   Структура данных

  1. Кнопка сохранить.
    1. При нажатии на данную кнопку происходит сохранение данных.
    2. Если поле "Дата" не равна текущей дате, то кнопка не активна и сохранение невозможно.
    3. Строки, которые были изменены и будут сохраняться подсвечены зеленым фоном.
  2. Проект и дата.
    1. Работа возможна только при выбранном проекте. При выборе проекта заполняется иерархический список Композиций и Комплектов данного проекта.
    2. По умолчанию, при открытии устанавливается текущая дата. Если дата равна текущей, то возможно сохранение изменений.
    3. При изменении Проекта или Даты Список Комплектов перегружается и все не сохраненные изменения пропадают.
  3. Ответственные по умолчанию.
    1. Ответственные подразделений по умолчанию. Изменение невозможно, можно открыть карточку сотрудника.
  4. Назначение ответственного.
    1. Изменять значения можно только в колонках «Менеджер», «Ведущий конструктор», «Дизайнер».
    2. Пользователь с «Полными правами» может изменять все три колонки.
    3. Пользователь с «Ведущий конструктор» может изменять колонку «Ведущий конструктор».
    4. Пользователь с правами «Дизайнер» может изменять колонку «Дизайнер».
  5. Структура данных.

    1. Данные комплекты выводиться в блоках объединенные Композиций.
    2. (4) Первый блок — это комплекты не входящие в Композицию. В данном блоке изменения ответственного производиться вручную в каждой строке. Строка в которой произошли изменения подсвечивается зеленым фоном.
    3. Далее идут блоки композиций:
      1. Первая строка внутри блока сама композиция
      2. Далее идут комплекты которые включены в эту композицию
      3. Можно изменять ответственного на каждый комплект вручную
      4. Изменение ответственного в композиции не изменяет ответственного в подчинённых комплектах.
      5. Для более быстрого изменения ответственного можно выбрать его в строчке (5). Данный ответственный будет назначен композиции. У композиции будет определен конструктор и то же ответственный будет назначен комплектом у которых совпадает конструктор с композицией.

Канбан ТЗ

Данная обработка служит для отображения данных по работе создания ТЗ дизайнерами.

  1. Блок Настройки.
    1. Поле проекты.
      1. Возможно добавление нескольких проектов. При закрытии канбана и при последующем открытии значение данного поля восстанавливается.
    2. Поле Сотрудник.
      1. Если данное поле пустое, то на канбане отображаются все задачи по ТЗ организации. В строке «Информация колонке», в графе «Всего задач» указывается количество карточек в каждой колонке. При не заполненом поле "Сотрудник" перемещение карточек невозможно.
      2. Если в данном поле указан дизайнер, то на канбане отображаются карточки с изделиями и композициями, с которыми работает указанный дизайнер. В строке «Информация колонке», в графе «Всего задач» указывается количество задач дизайнера без учета фильтров. Например, если у дизайнера две задачи, одна по проекту 01, а вторая 02 и в поле Проект выбран проект 01, то будет отображена одна карточка, но графе «Всего задач» будет число 2. При заполненом поле "Сотрудник" возможно перемещение карточек.
    3. Поле Поиск.
      1. Поиск осуществляется среди карточек, которые отображены на экране. Для начала поиска нужно в поле «Поиск» начать ввод текста, который входит в название карточки. Это не обязательно должен быть номер. Примеры текста для поиска: «31.104», «Камин», «Гостин» и т.д.
      2. По мере ввода текста поиска будет сокращать список карточек, которые соответствуют значению поиска
